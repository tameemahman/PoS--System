# PoS  System
 An E-Commerce point of sales system using SwingAnt Project in Apache

Front-End Development:
- Java Swing
- CSS
- JavaScript
- Bootstrap

Back-End Development:
- Java Swing
- MySQL

Software And Tools Required:
- MySQL
- MySQL Workbench
- Apache NetBeans
- Java JDK 17+
